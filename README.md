# Pdf-To-Text

## Overview
The Pdf-To-Text project is designed to extract text from PDF files efficiently. The goal is to provide an easy-to-use interface for users who need to convert PDF documents into plain text.

## Features
- Extracts text from multiple PDF formats.
- Batch processing capability.
- Simple command-line interface.
- Supports various languages.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dragoredempire/Pdf-To-Text.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pdf-To-Text
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application with the following command:
```bash
python pdf_to_text.py <path_to_pdf>
```
Replace `<path_to_pdf>` with the actual path of the PDF file you wish to convert.
